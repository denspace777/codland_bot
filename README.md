Это код телеграмм бота, главной фишкой которого являеться определение печенья по картинке.

Из прочих возможностей есть: приветствие и прощание с пользователем, генерация паролей (не используйте эту фишку для реальных паролей, я никакой ответственности не несу), делать случайный смайл, отправлять рандомную картинку утки
